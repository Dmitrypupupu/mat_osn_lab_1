# Пояснение к отчёту (index.typ)

Этот документ подробно объясняет каждый раздел файла `index.typ`: какие данные используются, какие формулы применяются, как именно производятся вычисления и что означают результаты.

---

## Исходные данные

**Выборка 1** — 50 чисел (вариант 20), используется для проверки гипотезы о нормальном распределении.

**Выборка 2** — первые 20 чисел из выборки 1, используется для оценки параметров и построения доверительных интервалов.

**Уровень значимости:** `α = 0.05`  
**Доверительная вероятность:** `P = 1 − α = 0.95`

---

## Задание 1. Проверка гипотезы о нормальном распределении

### Что проверяется

Выдвигается нулевая гипотеза **H₀**: генеральная совокупность подчиняется нормальному закону распределения.  
Альтернативная гипотеза **H₁**: распределение не является нормальным.

Проверка выполняется с помощью **критерия согласия Пирсона (χ²)** при `α = 0.05`.

---

### Шаг 1. Определение числа интервалов гистограммы (формула Стёрджесса)

```
k = 1 + 3.322 · lg(n)
```

Здесь `n = 50` — объём выборки, `lg` — логарифм по основанию 10.

```
k = 1 + 3.322 · lg(50) = 1 + 3.322 · 1.699 ≈ 6.64 → k = 7
```

Формула Стёрджесса даёт оптимальное число интервалов для гистограммы, чтобы распределение было видно достаточно подробно, но без излишнего «шума».

---

### Шаг 2. Ширина интервала гистограммы

```
h = (x_max − x_min) / k = (28.06 − 15.69) / 7 ≈ 1.77
```

Ширина интервала `h` равна размаху выборки, делённому на число интервалов.

---

### Шаг 3. Объединение малочисленных интервалов

Для корректного применения критерия Пирсона в каждом интервале должно быть не менее 5 наблюдений (`n_i ≥ 5`). Три правых интервала (с `n_i = 3`, `n_i = 5`, `n_i = 3`) объединяются в один. В итоге получается **5 объединённых интервалов**.

---

### Шаг 4. Выборочное среднее по серединам интервалов

Для расчёта параметров нормального распределения используется **взвешенное среднее по серединам** объединённых гистограммных интервалов (не по исходным данным):

```
x̄_в = (Σ m_i · x_i) / n
```

- `m_i` — число наблюдений в `i`-м интервале  
- `x_i` — середина `i`-го интервала  
- `n = 50`

```
x̄_в = (7·16.57 + 11·18.34 + 10·20.11 + 11·21.88 + 3·23.64 + 5·25.41 + 3·27.18) / 50
     = 1039.02 / 50 ≈ 20.780
```

> **Почему используются середины интервалов, а не исходные значения?**  
> Потому что гипотеза проверяется по сгруппированным данным гистограммы, а для расчёта параметров нормальной кривой используются именно эти группировки.

---

### Шаг 5. Выборочное среднеквадратическое отклонение по серединам интервалов

```
S_в = √( Σ m_i · (x_i − x̄_в)² / n )
```

Для каждого интервала вычисляется `m_i · (x_i − x̄_в)²`, результаты суммируются, делятся на `n` и берётся квадратный корень.

```
Σ m_i · (x_i − x̄_в)² = 461.961

S_в = √(461.961 / 50) = √9.239 ≈ 3.040
```

> Обратите внимание: здесь используется **делитель `n`** (а не `n−1`), т.к. это дисперсия, вычисленная по сгруппированным данным для параметрической кривой.

---

### Шаг 6. Вычисление теоретических частот (через функцию Лапласа)

Для каждого объединённого интервала `(a_i; b_i)` вероятность попадания в него значения при нормальном распределении N(x̄_в, S_в):

```
P_i = Φ(z_{b_i}) − Φ(z_{a_i}),   где z = (x − x̄_в) / S_в
```

`Φ(z)` — функция Лапласа (интеграл вероятности, в форме «от 0 до z» или «от −∞ до z» — зависит от таблицы; в отчёте используется форма с центральной симметрией):

- Для `z → −∞`: `Φ(z) = 0` (т.е. половина от 0.5, с поправкой на знак)
- Для `z → +∞`: `Φ(z) = 1`

Конкретные расчёты:

**Интервал 1 `(−∞; 17.46)`:**
```
z_{b_1} = (17.46 − 20.780) / 3.040 ≈ −1.09
P_1 = Φ(−1.09) = 0.5 − Φ₀(1.09) = 0.5 − 0.3621 = 0.1379
n'_1 = 50 · 0.1379 ≈ 6.90
```

**Интервал 2 `[17.46; 19.22)`:**
```
z_{a_2} = −1.09,  z_{b_2} = (19.22 − 20.780) / 3.040 ≈ −0.51
P_2 = Φ₀(1.09) − Φ₀(0.51) = 0.3621 − 0.1950 = 0.1671
n'_2 = 50 · 0.1671 ≈ 8.36
```

**Интервал 3 `[19.22; 20.99)`:**
```
z_{a_3} = −0.51,  z_{b_3} = (20.99 − 20.780) / 3.040 ≈ 0.07
P_3 = Φ₀(0.07) + Φ₀(0.51) = 0.0279 + 0.1950 = 0.2229
n'_3 = 50 · 0.2229 ≈ 11.15
```

**Интервал 4 `[20.99; 22.76)`:**
```
z_{a_4} = 0.07,  z_{b_4} = (22.76 − 20.780) / 3.040 ≈ 0.65
P_4 = Φ₀(0.65) − Φ₀(0.07) = 0.2422 − 0.0279 = 0.2143
n'_4 = 50 · 0.2143 ≈ 10.72
```

**Интервал 5 `[22.76; +∞)`:**
```
P_5 = 0.5 − Φ₀(0.65) = 0.5 − 0.2422 = 0.2578
n'_5 = 50 · 0.2578 ≈ 12.89
```

> **`Φ₀(z)`** — табличная функция Лапласа в «однопольной» форме (значения для `z > 0`), т.е. площадь кривой от 0 до z.

---

### Шаг 7. Вычисление наблюдаемого значения χ²

Для каждого интервала вычисляется компонента критерия Пирсона:

```
(n_i − n'_i)² / n'_i
```

где `n_i` — фактическое число наблюдений, `n'_i` — теоретическое.

Наблюдаемое значение критерия:

```
χ²_набл = Σ (n_i − n'_i)² / n'_i = 0.0015 + 0.8336 + 0.1186 + 0.0073 + 0.2770 ≈ 1.24
```

---

### Шаг 8. Число степеней свободы и вывод

```
k = s − 1 − r = 5 − 1 − 2 = 2
```

- `s = 5` — число объединённых интервалов  
- `r = 2` — число параметров, оцениваемых по выборке (среднее и СКО)

Критическое значение по таблице χ²-распределения при `α = 0.05` и `k = 2`:

```
χ²_крит = 5.99
```

Так как `χ²_набл = 1.24 < 5.99 = χ²_крит` → **нулевая гипотеза H₀ принимается**: данные не противоречат нормальному распределению.

## Задание 2. Несмещённые оценки параметров распределения

Используется **Выборка 2** (`n = 20` элементов).

### Несмещённая оценка среднего (выборочное среднее)

```
x̄ = (1/n) · Σ x_i = 395.20 / 20 = 19.760
```

Это несмещённая оценка: `E[x̄] = μ` (математическое ожидание генеральной совокупности).

---

### Несмещённая оценка дисперсии (исправленная выборочная дисперсия)

```
S² = (1 / (n − 1)) · Σ (x_i − x̄)²
```

Делитель `(n − 1)` вместо `n` делает оценку несмещённой: `E[S²] = σ²`.

```
Σ (x_i − x̄)² = 152.714

S² = 152.714 / 19 ≈ 8.038
```

---

### Несмещённая оценка среднеквадратического отклонения

```
S = √S² = √8.038 ≈ 2.835
```

> **Важно:** оценка `S` является лишь приближённо несмещённой для СКО; строго несмещённой является только оценка дисперсии `S²`.

---

## Задание 3. Доверительный интервал для среднего при **известной** дисперсии

**Условие:** дисперсия генеральной совокупности считается известной — берётся `S_в² ≈ 9.239` из выборки 1, т.е. `S_в ≈ 3.040`.

**Используемое распределение:** стандартное нормальное Z (так как дисперсия известна).

### Критическое значение

Из условия `2·Φ₀(t) = P = 0.95` → `Φ₀(t) = 0.475`:

```
t = 1.960   (по таблице функции Лапласа)
```

### Предельная ошибка выборки

```
Δ = t · S_в / √n = 1.960 · 3.040 / √20 = 5.958 / 4.472 ≈ 1.332
```

### Доверительный интервал

```
x̄ − Δ < μ < x̄ + Δ
19.760 − 1.332 < μ < 19.760 + 1.332
18.428 < μ < 21.092
```

С вероятностью 0.95 истинное среднее μ находится в интервале **(18.428; 21.092)**.

---

## Задание 4. Доверительный интервал для среднего при **неизвестной** дисперсии

**Условие:** дисперсия генеральной совокупности неизвестна — используется выборочная оценка `S = 2.835`.

**Используемое распределение:** распределение Стьюдента (t-распределение) с `k = n − 1 = 19` степенями свободы.

### Критическое значение

При `α = 0.05` и `k = 19`:

```
t_кр = 2.093   (по таблице Стьюдента)
```

### Предельная ошибка выборки

```
Δ = t_кр · S / √n = 2.093 · 2.835 / √20 = 5.934 / 4.472 ≈ 1.327
```

### Доверительный интервал

```
x̄ − Δ < μ < x̄ + Δ
19.760 − 1.327 < μ < 19.760 + 1.327
18.433 < μ < 21.087
```

С вероятностью 0.95 истинное среднее μ находится в интервале **(18.433; 21.087)**.

> **Сравнение с заданием 3:**  
> Интервал при неизвестной дисперсии (`Δ ≈ 1.327`) оказался незначительно уже, чем при известной (`Δ ≈ 1.332`). Это объясняется тем, что выборочное СКО `S ≈ 2.835` немного меньше `S_в ≈ 3.040`, что компенсирует «расширение» из-за использования t-распределения.

---

## Задание 5. Доверительный интервал для дисперсии

**Используемое распределение:** χ²-распределение с `k = n − 1 = 19` степенями свободы.

### Квантили χ²-распределения

При `α = 0.05` берутся двусторонние квантили уровней `α/2 = 0.025` и `1 − α/2 = 0.975`:

```
χ²_{0.025} = 8.907   (нижний квантиль)
χ²_{0.975} = 32.852  (верхний квантиль)
```

### Формула доверительного интервала для дисперсии

```
(k · S²) / χ²_{1−α/2}  <  σ²  <  (k · S²) / χ²_{α/2}
```

Числитель:

```
k · S² = 19 · 8.038 = 152.714
```

Подставляем:

```
152.714 / 32.852  <  σ²  <  152.714 / 8.907
4.648  <  σ²  <  17.146
```

С вероятностью 0.95 истинная дисперсия σ² находится в интервале **(4.648; 17.146)**.

> **Почему интервал несимметричен?**  
> χ²-распределение несимметрично (скошено вправо), поэтому доверительный интервал для дисперсии асимметричен относительно точечной оценки `S²`.
